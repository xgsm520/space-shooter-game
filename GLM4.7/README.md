# 太空飞机大战游戏 （AI）

## 简介
这是一个基于HTML5 Canvas的太空飞机大战游戏，玩家控制飞机在太空中射击敌人，躲避碰撞，并收集奖励。游戏包含丰富的视觉效果、音效和多种游戏机制。（AI生成）

## 功能特色

### 🎮 游戏机制
- **双控支持**：支持键盘WASD控制和移动端触摸控制
- **自动射击**：玩家飞机自动发射子弹攻击敌人
- **碰撞检测**：精确的碰撞检测系统

### 💔 生命系统
- **生命值**：初始生命值为100点
- **伤害机制**：飞机敌人造成15点伤害，陨石敌人造成10点伤害
- **游戏结束**：生命值为0时游戏结束
- **视觉反馈**：被击中时飞船闪烁红色

### 🎁 奖励系统
- **随机掉落**：击败敌人有30%概率掉落奖励
- **智能分配**：飞机敌人更可能掉落武器（80%），陨石敌人更可能掉落治疗（70%）
- **治疗奖励**：恢复25点生命值（最多100点）
- **武器升级**：提升武器等级，解锁更多子弹模式

### 🔫 武器系统
- **等级1**：单发子弹
- **等级2**：双发子弹
- **等级3**：扇形三发子弹
- **等级4**：四发子弹
- **等级5**：五发扇形子弹
- **限时效果**：武器效果持续5秒后恢复到1级

### 🎵 音效系统
- **Web Audio API**：使用Web Audio API生成各种音效
- **多样音效**：射击、爆炸、受伤、拾取奖励等音效
- **背景音乐**：循环播放的太空主题音乐

### 📱 移动端适配
- **触摸控制**：支持手指触摸控制飞机移动
- **视觉引导**：触摸时显示目标指示器
- **响应式设计**：适配不同屏幕尺寸

### ⚙️ 游戏平衡
- **难度递增**：随着分数增加，敌人生成频率提高
- **多样化敌人**：飞机和陨石两种敌人类型
- **策略选择**：在攻击和收集奖励之间做策略选择

## 操作说明

### PC端
- **W/A/S/D**：控制飞机移动
- **开始游戏**：点击"开始游戏"按钮
- **暂停/继续**：点击相应按钮
- **重新开始**：游戏结束后点击"重新开始"

### 移动端
- **触摸拖拽**：触摸屏幕拖拽控制飞机移动
- **开始游戏**：点击"开始游戏"按钮
- **暂停/继续**：点击相应按钮
- **重新开始**：游戏结束后点击"重新开始"

## 游戏玩法
1. 控制飞机在屏幕上移动，避开敌人
2. 游戏会自动发射子弹消灭敌人
3. 击败敌人获得分数，有机会获得奖励
4. 拾取治疗奖励恢复生命值
5. 拾取武器奖励提升攻击力
6. 避免与敌人碰撞，否则会损失生命值
7. 生命值为0时游戏结束
